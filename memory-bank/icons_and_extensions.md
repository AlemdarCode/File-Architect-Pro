# Icon Listesi ve İşlevleri

## Mevcut Iconlar (icons/ klasörü)

| Icon Dosyası | Kullanım Yeri | İşlev |
|--------------|---------------|-------|
| `folder-solid.svg` | Klasör (Kapalı) | Kapalı klasör gösterimi |
| `folder-open-solid.svg` | Klasör (Açık) | Açık klasör gösterimi |
| `file-lines-solid.svg` | Dosya | Genel dosya gösterimi |
| `file-image.svg` | Resim Dosyası | jpg, png, gif, bmp, svg, webp |
| `file-audio.svg` | Ses Dosyası | mp3, wav, flac, m4a, ogg |
| `xmark-solid.svg` | Silme Butonu | Dosya/klasör silme (X ikonu) |

## Yaygın Uzantılar (Uzantı Filtresi İçin)

### Resim
- jpg, jpeg, png, gif, bmp, svg, webp, ico, tiff

### Ses
- mp3, wav, flac, m4a, ogg, aac, wma

### Video
- mp4, avi, mkv, mov, wmv, flv, webm

### Belge
- pdf, doc, docx, xls, xlsx, ppt, pptx, odt, ods, odp

### Metin
- txt, md, json, xml, html, css, js, py, java, c, cpp, h

### Arşiv
- zip, rar, 7z, tar, gz, bz2

### Çalıştırılabilir
- exe, msi, bat, cmd, sh
